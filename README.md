# React-native
## Install environments
**Công cụ (Tool):**<br>
* Viual Studio Code<br>
* Expo Go

**Ngôn ngữ (Language):**<br>
* Javascript

Install npm:
```
brew install npm
```
Install expo:
```
npm install -g expo-cli
```
or
```
npm install --global expo-cli
```
Open project. Run cmd để get QR code:
```
npm run ios
```
